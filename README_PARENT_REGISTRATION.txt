ระบบลงทะเบียนผู้ปกครองและผูกนักเรียนอัตโนมัติ เวอร์ชัน 1.4.0

หลักการทำงาน
1. โรงเรียนสร้างรหัสเชื่อมบัญชีเฉพาะนักเรียน
2. ผู้ปกครองกรอกชื่อ อีเมล เลขประจำตัวนักเรียน รหัสเชื่อม และตั้งรหัสผ่าน
3. Supabase ส่งอีเมลยืนยัน
4. ผู้ปกครองกดยืนยันอีเมล แล้วเข้าสู่ระบบ
5. ระบบผูกนักเรียนให้อัตโนมัติเมื่อเข้าสู่ระบบครั้งแรก

ขั้นตอนติดตั้ง

1. เปิด Supabase > SQL Editor
2. เปิด parent_self_registration_setup.sql
3. คัดลอกทั้งหมดไปวางและกด Run
4. ไปที่ Authentication > Sign In / Providers > Email
5. เปิด Allow new users และ Confirm email
6. ไปที่ Authentication > URL Configuration
7. ตั้ง Site URL:
   https://klunschool.github.io/school-app/
8. เพิ่ม Redirect URL:
   https://klunschool.github.io/school-app/**
9. ตั้ง Custom SMTP ก่อนใช้งานจริง
10. อัปโหลด index.html และ sw.js ทับใน GitHub

สำคัญเรื่องการส่งอีเมล

Supabase SMTP เริ่มต้นใช้สำหรับทดสอบเท่านั้น ส่งได้จำกัดมาก
และอาจไม่ส่งให้อีเมลที่ไม่ใช่สมาชิกทีมโครงการ
สำหรับผู้ปกครองจริงควรตั้ง Custom SMTP เช่น Resend, Brevo,
SendGrid, Postmark หรือ Amazon SES ที่ Authentication settings

การสร้างรหัสเชื่อมทั้งห้อง

ตัวอย่าง ป.6/1:

select *
from public.create_parent_codes_for_class('ป.6/1');

หลัง Run ให้ Export ผลลัพธ์ทันที เพราะฐานข้อมูลเก็บเฉพาะค่า hash
และไม่สามารถเปิดดูรหัสเดิมภายหลังได้

การสร้างรหัสเชื่อมรายคน

select *
from public.create_parent_code_for_student('เลขประจำตัวนักเรียน');

ควรส่งรหัสเชื่อมให้ผู้ปกครองเป็นการส่วนตัว
ห้ามโพสต์รายชื่อ เลขประจำตัว และรหัสเชื่อมในกลุ่มสาธารณะ

ข้อความอีเมลยืนยัน

Supabase > Authentication > Emails > Confirm signup

Subject:
ยืนยันการลงทะเบียนแอปโรงเรียนกลั่นวิทยา

Body:
เรียน ผู้ปกครอง

โรงเรียนกลั่นวิทยาได้รับคำขอลงทะเบียนแอปโรงเรียน
กรุณากดปุ่มด้านล่างเพื่อยืนยันอีเมลของท่าน

<a href="{{ .ConfirmationURL }}">ยืนยันอีเมลและเปิดใช้งานบัญชี</a>

หลังยืนยันแล้ว กรุณากลับเข้าสู่แอปด้วยอีเมลและรหัสผ่านที่ท่านตั้งไว้
ระบบจะเชื่อมข้อมูลบุตรหลานให้อัตโนมัติ

หากท่านไม่ได้เป็นผู้ลงทะเบียน กรุณาไม่ต้องดำเนินการใด ๆ
