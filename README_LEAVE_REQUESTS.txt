ระบบใบลา เวอร์ชัน 1.3.0

ติดตั้งตามลำดับ:

1. เปิด Supabase > SQL Editor
2. เปิดไฟล์ leave_requests_setup.sql
3. คัดลอกโค้ดทั้งหมด วางใน SQL Editor และกด Run
4. อัปโหลด index.html และ sw.js ทับไฟล์เดิมใน GitHub
5. รอ GitHub Pages อัปเดต แล้วเปิดแอปใหม่

การใช้งาน:

- ผู้ปกครอง: หน้าหลัก > แจ้งลา
- ครูประจำชั้น: หน้าหลัก > รับใบลา
- ครูเห็นเฉพาะนักเรียนในห้องที่ตนเป็นครูประจำชั้น
- ผู้ดูแลระบบเห็นใบลาทุกห้อง
- เมื่อครูอนุมัติ ระบบจะเพิ่มสถานะ "ลา" ในตารางเช็คชื่อของวันนั้น
- ผู้ปกครองดูสถานะ รอตรวจสอบ / อนุมัติแล้ว / ไม่อนุมัติ ได้จากประวัติการลา

หมายเหตุ:

- ใบลาที่เคยส่งก่อนติดตั้งระบบนี้จะยังอยู่ในตาราง attendance แบบเดิม
  และจะไม่ปรากฏในประวัติใบลาชุดใหม่
