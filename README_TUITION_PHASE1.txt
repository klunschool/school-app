ระบบค่าเทอม เฟส 1

1. เปิด Supabase > SQL Editor
2. เปิดไฟล์ supabase_tuition_phase1.sql แล้วคัดลอกทั้งหมดไปวางและกด Run
3. กำหนดบัญชีฝ่ายการเงิน โดยแก้ email แล้วรันคำสั่งนี้:

update public.profiles
set role = 'finance'
where id = (
  select id from auth.users
  where lower(email) = lower('อีเมลฝ่ายการเงิน')
);

4. อัปโหลด index.html และ sw.js ทับไฟล์เดิมใน GitHub
5. รอ GitHub Pages อัปเดต แล้วกด Ctrl + Shift + R หนึ่งครั้ง

การทดลอง
- ฝ่ายการเงิน: เปิดเมนูค่าเทอม เลือกภาคเรียน ห้อง รายการ จำนวนเงิน และวันครบกำหนด
- ผู้ปกครอง: เลือกบุตร > ค่าเทอม > แจ้งชำระเงิน > แนบสลิป
- ฝ่ายการเงิน: เปิดค่าเทอม > ดูสลิป > อนุมัติ
- ระบบจะเปลี่ยนสถานะและสร้างเลขใบเสร็จอัตโนมัติ
