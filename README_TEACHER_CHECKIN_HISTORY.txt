ประวัติสแกนเฉพาะห้องประจำชั้น เวอร์ชัน 1.7.0

ขั้นตอนติดตั้ง
1. เปิด Supabase > SQL Editor
2. เปิดไฟล์ teacher_checkin_history_setup.sql
3. คัดลอกทั้งหมด วาง แล้วกด Run
4. อัปโหลด index.html และ sw.js ทับไฟล์เดิมใน GitHub
5. รอ GitHub Pages Deploy แล้วเปิดแอปใหม่

สิทธิ์การดูข้อมูล
- ครู: เห็นประวัติสแกนเฉพาะนักเรียนในห้องที่ผูกไว้ใน teacher_class
- ครูที่มีมากกว่าหนึ่งห้อง: เห็นเฉพาะห้องที่ผูกกับบัญชีครูทั้งหมด
- Admin: เห็นประวัติทุกห้อง
- ผู้ปกครอง: เห็นเฉพาะบุตรหลานที่ผูกบัญชี

หากครูไม่เห็นข้อมูล
ตรวจสอบว่าอีเมลครูถูกผูกกับห้องในตาราง teacher_class แล้ว
